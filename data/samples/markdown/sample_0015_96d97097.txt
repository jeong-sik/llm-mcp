**GET /users/{id}**

Retrieve a user by ID

### Description

This endpoint returns a single user based on the provided ID.

### Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| id | integer | Yes | The unique identifier of the user to retrieve |

### Examples

#### Example Request 1: Successful retrieval

```bash
GET /users/123 HTTP/1.1
Host: example.com
Accept: application/json
```

#### Example Response 1:

```json
{
    "id": 123,
    "name": "John Doe",
    "email": "john.doe@example.com"
}
```

#### Example Request 2: User not found

```bash
GET /users/999 HTTP/1.1
Host: example.com
Accept: application/json
```

#### Example Response 2:

```json
{
    "error": {
        "code": 404,
        "message": "User not found"
    }
}
```

#### Example Request 3: Bad request (invalid ID)

```bash
GET /users/abc HTTP/1.1
Host: example.com
Accept: application/json
```

#### Example Response 3:

```json
{
    "error": {
        "code": 400,
        "message": "Invalid user ID"
    }
}
```
